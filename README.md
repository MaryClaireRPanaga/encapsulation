# Encapsulation Project

A Python application designed to demonstrate the core principles of Object-Oriented Programming (OOP), specifically focusing on Encapsulation. This project showcases how to restrict direct access to methods and variables to prevent the accidental modification of data, while utilizing public methods (getters and setters) to safely inspect and alter state.

## 🚀 Features
- Data Hiding: Uses private or protected attributes to secure sensitive class data.
- Controlled Access: Implements getter and setter methods to validate and manage data modification.
- Robust Logic and Error Handling: Prevents invalid inputs or logical errors during execution to keep the program running smoothly.
- Clean OOP Architecture: Built completely using modular class-based programming principles.

## 🛠️ Built With
- Language: Python 3.x
- Version Control: Git and GitHub
- Development Environment: VS Code

## 📁 Code Structure
The project is organized to separate object logic from execution:
- main.py — The primary entry point that handles user interaction and demonstrates object creation.
